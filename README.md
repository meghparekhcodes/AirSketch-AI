# 🎨 AirSketch AI

Draw in the air using just your hand! **AirSketch AI** is an AI-powered computer vision project that transforms your index finger into a virtual pen. Using your webcam and MediaPipe Hand Tracking, you can create beautiful drawings without touching a screen or using a mouse.

---

## 🌐 Live Demo

🚀 **Try AirSketch AI in your browser:**

**👉 https://meghparekhcodes.github.io/AirSketch-AI/**

> **Note:** Allow camera access when prompted. The application runs entirely in your browser, and no video is uploaded or stored.

---

## ✨ Features

- 🖐️ AI-powered real-time hand tracking
- ✏️ Draw using pinch gestures
- 🎨 Multiple neon drawing colors
- 💾 Save drawings as PNG
- 🧹 One-click canvas clearing
- 📷 Live webcam interface
- ⚡ Smooth and responsive drawing
- 🌐 Fully browser-based
- 🔒 Privacy-friendly (all processing happens locally)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- MediaPipe Hands
- HTML5 Canvas API
- WebRTC (Camera Access)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/meghparekhcodes/AirSketch-AI.git
```

### Open the project

Simply open **index.html** in your browser, or visit the live demo.

No installation or backend setup is required.

---

## 🎮 How to Use

1. Open the application.
2. Click **Activate Camera**.
3. Allow webcam permission.
4. Pinch your **thumb and index finger** together to start drawing.
5. Release the pinch to stop drawing.
6. Choose different neon colors.
7. Clear the canvas anytime.
8. Save your artwork as a PNG image.
