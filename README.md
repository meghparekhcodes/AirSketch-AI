# 🎨 AirSketch AI

Draw in the air using AI-powered hand tracking! AirSketch AI transforms your index finger into a virtual pen, allowing you to create drawings in real time using just your webcam.

👉 **Try it here:**

## ✨ Features

- 🖐️ Real-time hand tracking
- ✏️ Draw using your index finger
- 🎨 Multiple neon drawing colors
- 🧹 Clear canvas option
- 💾 Save artwork as PNG
- 📷 Live webcam interface
- ⚡ Smooth and responsive drawing
- 🤖 Powered by MediaPipe Hands
