# 🎨 Air Drawer

Draw in the air using just your hand! **Air Drawer** is a computer vision project that transforms your finger into a virtual pen. Using a webcam and AI-powered hand tracking, you can create drawings without touching a screen or using a mouse.

---

## ✨ Features

- 🖐️ Real-time hand tracking
- ✏️ Draw using your index finger
- 🎨 Multiple drawing colors
- 🧹 Clear canvas option
- 📷 Live webcam interface
- ⚡ Smooth and responsive drawing experience
- 🤖 Powered by AI hand detection

---

## 🛠️ Tech Stack

- **Python**
- **OpenCV**
- **MediaPipe**

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/meghparekhcodes/Air-Drawer.git
```

### 2. Go to the project folder

```bash
cd Air-Drawer
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python air_drawer.py
```

---

## 🎮 How to Use

1. Launch the application.
2. Allow access to your webcam.
3. Raise your index finger.
4. Move your finger in the air to draw.
5. Use the available gestures/buttons to:
   - Change colors
   - Clear the canvas
6. Enjoy drawing without touching any device!
